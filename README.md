# 頼みごとを優しく、丁寧な言葉遣いで、まとめてくれるLINEBot

# 概要：
このプロダクトは、人が頼みごとをする際に、ネガティブな感情や言葉遣いを使わずに、優しく丁寧な言葉遣いでまとめることを目的としたLINEbotです。

# 始めた理由：
というのも頼みごとをする際に、それが攻撃的なものになることを最小限に抑えたいと思ったからです。
